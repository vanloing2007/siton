siton is designed for users without programming skills. Its interface guides you through building and testing strategies.

### 1. Opening siton

- Find the siton icon on your desktop or in the Start menu.  
- Double-click the icon to open the app.

- Use the menu or buttons to add indicators from the list of 25+ available options.
